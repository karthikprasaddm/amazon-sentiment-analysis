# Amazon Product Reviews Sentiment Analysis

A 5-class sentiment analysis system built using Natural Language 
Processing (NLP) and Machine Learning to classify Amazon product 
reviews into Very Negative, Negative, Neutral, Positive, and 
Very Positive categories.

## Overview

This project builds a complete end-to-end text classification 
pipeline that preprocesses raw customer review text, extracts 
features using TF-IDF Vectorization, trains multiple ML algorithms, 
and predicts sentiment with confidence scores in real-time.

## Features

- 5-class sentiment classification system
- Complete NLP text preprocessing pipeline
- TF-IDF feature extraction with optimized parameters
- Trains and compares 5 ML algorithms automatically
- Saves best model for reuse
- Real-time interactive prediction with confidence scores
- Probability distribution output for each prediction
